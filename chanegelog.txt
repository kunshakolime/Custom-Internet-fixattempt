## Changelog
v1.0 → v1.1
- Fix SSH banner consumed by read_headers() on second response call
- Add _is_upgrade_response() to recognise both 101 and 200+Upgrade as terminal

v1.1 → v1.2
- Add [split] as a block separator for multi-request payloads
- Drain intermediate response bodies (chunked + Content-Length) before sending next block
- Raise ConnectionError immediately if all blocks exhausted without upgrade

v1.2 → v2.0
- Replace config.py with a configs/ folder; main.py auto-selects if one file present, otherwise prompts numbered menu at startup
- Add --socks flag to main.py for SOCKS-only mode (no routing changes, cross-platform, no admin required)
- Integrate tap.py into main.py: hosts are resolved once in main.py and passed as pre-resolved IPs into a new run_tap() entry point
- Remove standalone entrypoint, internal resolve_host(), CONFIG import, and EXTRA_BYPASS_HOSTS from tap.py; it is now a library module only
- start_tun2socks() now accepts socks_port as a parameter instead of reading from CONFIG

v2.0 → v2.1
- Fix UnicodeDecodeError on non-UTF-8 Windows locales by adding errors="replace" to all subprocess.run(text=True) calls in tap.py
- Guard all result.stdout accesses in tap.py against None before calling splitlines()
- Deduplicate netsh interface query into _netsh_interfaces() helper used by both raise_other_interface_metrics() and restore_interface_metrics()
- Remove redundant second PowerShell call in wait_for_tun2socks_interface() that fetched adapter name separately from ifIndex
- Remove SSH_PORT from config and entire call chain; socket is already connected before Paramiko sees it so the port was never used
- Replace timeout-based _recv_all() in ssh_connector.py with _recv_exactly(n); fixes fragile SOCKS4 handshake parsing on slow connections
- Remove dead ver variable and slow bytes([x]) concat loop in _handle_socks4()
- Remove unused ssh_port attribute from SSHOverWebSocket.__init__()
- Clean up start_ssh_transport() by removing stale commented-out host key block
- Switch configs from .py to .toml; eliminates __pycache__ and removes importlib dependency
- Collapse PROXY_HOST+PROXY_PORT and TARGET_HOST+TARGET_PORT into single host:port strings in config; split once in main.py
- Suppress Paramiko channel failure log noise via logging.getLogger("paramiko").setLevel(CRITICAL)
- Add set_keepalive(60) on SSH transport to detect dead connections early
- Silence noisy [*] Opening SSH channel print; catch ChannelException in _open_ssh_channel to fail cleanly per channel

v2.1 → v3.0
- Replace Go-based tun2socks with badvpn-tun2socks for native udpgw support
- Add udpgw key to toml config; pass through main.py → run_tap() → start_tun2socks()
- Fix badvpn Windows --tundev format: tap0901:<adapter>:<TAP-IP>:<network>:<netmask>
- Add --udpgw-remote-server-addr flag to badvpn launch when udpgw is configured
- Remove SOCKS5 UDP ASSOCIATE handling from ssh_connector.py; UDP is now handled entirely by badvpn+udpgw without touching the SOCKS server
